# Module_11_Challenge

Plotting trends and using Prophet to forecast search statistics and sales.

* [hvplot](https://github.com/holoviz/hvplot)
* [Prophet](https://github.com/facebook/prophet)
* [NumPy](https://github.com/numpy/numpy)
